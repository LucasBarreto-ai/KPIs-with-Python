Este projeto tem como objetivo desenvolver uma aplicação de análise e exibição de KPIs (Key Performance Indicators) utilizando Python, com foco em tomada de decisão baseada em dados.

A solução realiza o processamento, limpeza e análise de dados, transformando informações brutas em indicadores estratégicos, como receita, lucro, margem, ticket médio, CAC, LTV e métricas de clientes. 
Os KPIs são calculados de forma automatizada e apresentados de maneira clara e estruturada, permitindo análises rápidas e confiáveis.

Objetivo do Projeto

Demonstrar habilidades em análise de dados, lógica de negócios e uso de Python para geração de insights, sendo ideal para compor um portfólio profissional de Data Analytics.
